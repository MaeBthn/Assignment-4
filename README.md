# Assignment 4

This repository contains the implementation for **Assignment 4** of the COMP 3505 course.  
The project was developed using **Java and Maven** and follows a modular structure for implementing the required classes and functionality.

## Project Overview

The goal of this assignment is to design and implement a program using object-oriented principles and appropriate data structures. The project focuses on building the required classes and methods while maintaining clean and organized code.

The project is built as a **Maven project**, which allows easy dependency management and consistent project structure.

## Technologies Used

- Java 17
- Apache Maven
- Object-Oriented Programming principles
- Git / GitHub for version control
